# Industrial Machine Vision – Foreign Object Detection in Oil Cans

This project focuses on automated visual inspection of oil cans using an industrial machine vision system.

The system was developed to detect foreign objects inside oil containers during the production process, such as insects, debris, or other contaminants that may affect product quality and safety.

The inspection system uses a Basler industrial camera and MVTec HALCON to capture and analyze images of the interior of the cans in real-time.

### Key Features
- Automated inspection of oil cans on a production line
- Detection of foreign objects (insects, debris, contaminants)
- Image acquisition using Basler industrial cameras
- Image processing and defect detection using MVTec HALCON
- Designed for industrial quality control applications

### Technologies
- Basler Industrial Cameras
- MVTec HALCON
- Industrial Machine Vision
- Image Processing for Quality Inspection

### Application
Industrial food production lines where automated visual inspection is required to ensure product safety and quality.
